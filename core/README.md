# Resume Analyzer (Django)

A web-based ATS system that analyzes resumes and provides:

## Features
- ATS Score Calculation
- Skill Detection
- Role Prediction
- Section Validation
- Resume History

## Tech Stack
- Python
- Django
- HTML/CSS

## How to Run
pip install django
pip install PyPDF2
pip install pdfplumber
pip install -r requirements.txt  
python manage.py runserver
